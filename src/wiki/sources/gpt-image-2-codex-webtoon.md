# 소스놀이터 — GPT 이미지 2.0 코덱스 웹툰 자동화 시연

## 개요
소스놀이터 채널의 GPT 이미지 2.0 실전 시연 영상 (6분 49초). 코덱스 CLI를 활용하여 웹툰 자동 제작, AI 변천사 타임라인 이미지 생성, 3D 설계도 변환, 메인보드 비교 분석까지 수행. GPT 이미지 2.0의 한국어 렌더링 정확도가 나노바나나 프로를 압도하며, 코덱스의 자율 에이전트 능력으로 전 과정이 자동화됨.

## 핵심 내용

### 1. 코덱스 CLI 무제한 이미지 설정
코덱스의 토큰을 사용해 이미지를 생성하므로, 5시간 주간 한도 내에서 이미지를 무제한으로 사용 가능. 설정 방법:
1. 코덱스 실행 후 config 파일 열기
2. `features` 항목의 `image_generation`을 `true`로 설정
3. 자연어로 부탁만 하면 됨

**중요**: 마지막에 반드시 "dall-e image"라고 적어야 이미지 생성이 작동함.

### 2. GPT 5.4 Fast 모델 선택
- 토큰 소모는 2배이지만 속도가 1.5배
- 빠른 결과가 필요한 경우에 적합
- 정확도와 품질은 동일

### 3. 웹툰 자동화 실전
코덱스에 지시: "등장 인물 일관성을 유지하고 주제는 바이브 코딩으로 스토리를 만들어 제작"
- 코덱스가 스스로 작업 진행 → 사용자는 대기만
- 캐릭터 일관성 유지 + 한국어 정확
- 뷰어까지 자동 생성
- 배경 속 한국어 디테일까지 완벽

### 4. AI 변천사 타임라인 생성
- 웹 조사 수행: 공식 릴리스 노트와 반응 조사
- 연구 결과를 이미지로 변환
- 정확한 연도 표기 + 한국어 철자 완벽
- "혼자서 일주일 꼬박해도 이런 퀄리티가 안 나온다"

### 5. 멀티모달 이미지 분석 — 메인보드 비교
코덱스의 분석 프로세스:
1. 메인보드 사진 2장 입력
2. 로고와 칩셋 표기 영역을 크롭하여 확인
3. 웹 조사로 공식 사양과 대조
4. 제조사와 모델명 식별
5. 이미지에 글자를 씌워 상세 설명
6. 핵심 비교표 + 장단점 정리

### 6. 나노바나나 프로 vs ChatGPT 이미지 비교

| 항목 | 나노바나나 프로 | ChatGPT (GPT 이미지 2.0) |
|------|----------------|--------------------------|
| 한국어 정확도 | 높음 | 완벽 (오타 제로) |
| 워터마크 | 있음 (제거 필요) | 없음 |
| 16:9 비율 | 지원 | 지원 (신규) |
| 해상도 | 높음 | 상대적으로 낮음 |
| 캐릭터 일관성 | 좋음 | 좋음 |

### 7. 실무 적용 분야
- 기업: 제품 설계도, 비교 분석 자료
- 연구소: 기술 분석 인포그래픽
- 광고: 시각 콘텐츠 제작
- 유튜브: 영상/썸네일 (16:9 비율 필수)
- 교육: AI 변천사, 타임라인 차트
