# 크로마키 이미지 프롬프트

## 개요
이미지 생성 시 순수 크로마키 그린 배경을 지정하는 프롬프트 템플릿. 배경 분리(matting) 작업을 위해 사용.

## 핵심 내용
- 프롬프트: `Solid chroma key green background(0, 255, 0)`
- RGB (0, 255, 0) = 순수 그린
- 용도: 생성된 이미지에서 배경을 쉽게 분리하기 위한 전처리

## 원문 인용
> Solid chroma key green background(0, 255, 0)
