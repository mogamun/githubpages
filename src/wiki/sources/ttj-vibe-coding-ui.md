# 투더제이 — 바이브코딩 결과물이 내 것 같지 않은 이유

## 개요
바이브 코딩으로 만든 결과물이 만족스럽지 않은 두 가지 이유와 해결책. 클론 코딩의 중요성, AI 디자인을 사람처럼 바꾸는 10가지 UI 프롬프트 패턴.

## 핵심 내용
- **이유 1**: 결과물 미명확 → 클론 코딩으로 기획력 확보
- **이유 2**: 이해 부족 → 컨트롤 불가 → UI/UX 개념 학습 필요
- **10가지 UI 프롬프트**: FontAwesome, Pretendard, 스크롤 애니메이션, 글래스모피즘, 벤토 레이아웃, 스켈레톤 UI, 3D 틸트, 카운팅 애니메이션, 햄버거 버튼, 시맨틱 HTML

## 원문 인용
> "이해를 못 했고, 그래서 컨트롤을 못 하기 때문이다"
