# GeekNews Weekly #352

## 개요
2026-03-30 ~ 2026-04-05 주요 뉴스. 에디토리얼에서 개인 지식 저장소의 진화사와 LLM Wiki 패러다임을 다룸. Claude Code 소스 유출, Gemma 4, Qwen 3.5/3.6, Cursor 3 등.

## 핵심 내용

### 에디토리얼: 지식 저장소 진화
종이 → Zettelkasten → PC/검색 → Evernote/Second Brain → Obsidian/양방향 링크 → LLM Wiki. RAG(검색 중심)에서 LLM Wiki(지식 재구성 중심)로의 패러다임 전환.

### 주요 뉴스
1. **LLM-Wiki**: Karpathy의 개인 지식 저장소 접근
2. **Claude Code 소스 유출**: npm 소스맵 파일 노출, 내부 구조 공개
3. **Claude Code 내부**: anti-distillation 가짜 도구, undercover 모드, KAIROS 자율 모드
4. **Claude Code 기능**: `/loop`(1주일 자동 반복), `/batch`(병렬 처리), `--teleport`
5. **Gemma 4**: Gemini 3 기반, 4개 크기, 140개 언어
6. **Qwen3.5-Omni**: 완전 옴니모달, 256k 컨텍스트
7. **Qwen3.6-Plus**: 에이전트형 코딩, 1M 컨텍스트
8. **Cursor 3**: 로컬+클라우드 에이전트 통합 워크스페이스
9. **Caveman**: 원시인 말투로 65~75% 토큰 절감
10. **1-bit Bonsai**: 최초 엔드투엔드 1비트 LLM, 1.15GB

## 원문 인용
> "RAG가 매번 질문할 때마다 원문에서 정보를 다시 찾아 조합하는 방식이라면, LLM Wiki는 복리형 구조"
