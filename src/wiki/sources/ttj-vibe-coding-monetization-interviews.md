# 투더제이 — 바이브 코딩으로 부업에서 본업까지

## 개요
4명의 코딩 수익화 성공 사례 인터뷰 (2시간 11분). 대리기사, 비전공자, 마케터, 50대 등 서로 전혀 다른 배경을 가진 사람들이 어떻게 코딩을 시작했고, 어떤 방식으로 수익을 만들었으며, 왜 코딩이 인생의 새로운 기회가 되었는지를 현실적으로 담은 영상. 투더제이 코딩 클래스 수강생들의 실제 경험담.

## 핵심 내용

### 인터뷰 1: 코딩으로 인생이 바뀐 대리기사 (01:30~44:54)
- **배경**: 30대 후반 남성. 학창시절 공부를 잘 못해 20대에는 현장일과 공장 일을 했고, 30대에는 대리운전과 배달을 병행. 컴퓨터와 완전히 무관한 직업
- **코딩 인식**: "코딩은 일 지식 분야 중에서도 레벨이 가장 높은 학문이라고 생각해서 절대 못 할 거라고 생각했다. 하더라도 중학교나 고등학교에 시작해야 하고 똑똑한 사람들만 할 수 있는 거라고 인식"
- **시작 계기**: 결혼한 지인의 배우자가 집에서 코딩만으로 사업하며 돈을 버는 것을 보고 자극. HTML/CSS를 유튜브에서 공부하다 CSS에서 "이건 내 길이 아니다"하고 포기. 이후 반복적으로 도전하고 포기
- **TTJ 선택 이유**: 유튜브 알고리즘으로 발견. "투더제이 님은 코딩을 어렵게 알려주는 느낌이 아니라 코딩으로 센스 있게 돈을 많이 버는 능력 있는 형 같은 느낌". 다른 강의들은 문법 위주라 어려웠으나 TTJ는 실용적 접근
- **현재**: 수강 후 빠른 성장 → 투더제이 코딩 클래스에서 튜터로 활동 중. 전업 대리기사에서 코딩 튜터로 전환 성공

### 인터뷰 2: 반년 만에 수익파이프 n개 만든 비전공자 (44:56~01:10:59)
- **배경**: 비전공자. 코딩 경험 전무
- **수익 구조**:
  - 자동화 수익: 월 300만 원 중반
  - 외주 수익: 800만 원 (수강생 도움)
  - 비용: Claude/GPT/Cursor AI 한 달에 20달러 외 비용 거의 없음
- **블로그 자동화 상세**:
  - 무료 API를 프롬프트로 잘 다듬어 "사람이 쓴 것처럼" 자동화
  - 여러 아이디를 돌리면서도 API 비용이 발생한 적이 없음
  - 수강생 중 1,000개 이상 아이디를 돌리는 분도 API 비용 zero
  - 워드프레스 운영 시 도메인+서버 비용 약 15,000~19,000원
- **커스터마이징의 중요성**: 수강생이 강의 코드 그대로 돌린 4개 블로그 중, 커스터마이징한 것은 하루 2,100명 방문자. 그대로 돌린 나머지 3개는 하루 50명. "커스터마이징만 조금 하면 수익화 잘 되는데 어려워하는 분들은 이걸 안 하신다"
- **세팅 효과**: 세팅 안 할 때 월 30~50만 원 → 이틀 세팅 후 바로 그 달 120만 원. 이후 자동화 안 돌려도 잔여 글로 월 60~70만 원 유지
- **수익화 어려워하는 분들의 공통점**: 코딩을 배워서 커스터마이징하는 방법을 배우는데도, 다 같이 똑같은 이미지와 글 구조로 올림. 차별화 없으면 수익 없음

### 인터뷰 3: 블로그 자동화로 주 300만원 버는 마케터 (01:11:01~01:29:47)
- **배경**: 기존 마케팅 업무 경험. 코딩과 마케팅의 결합
- **프로젝트 이력**:
  - 쿠팡 상품 정보 크롤링 프로그램 (크몽 판매, 약 8회 판매)
  - 엔사 카페 쿠팡 파트너스 자동 포스팅 프로그램
  - 대량화 작업 의뢰로 180만 원 수익
  - 현재: 블로그 자동화가 주 수입원. 하루 9,000~15,000개 글 자동 작성
- **수익**: 주 100~300만 원 (월 평균 500~600만 원). 3개월간 약 1,700만 원
- **들쭉날쭉 이유**: 마케팅 플랫폼(엔사) 불안정, 계정 문제, 블로그 폐쇄 등
- **성공 비결**: "운도 좋았지만, 무엇보다 두려워하지 않고 시도하는 것. 14시간 동안 밥도 안 먹고 문제를 파고든 끈기. 해결 안 되는 프로젝트는 과감히 드랍"
- **비전공자 조언**: "요즘은 AI가 코딩을 너무 잘해서 아이디어만 있으면 다 만들 수 있다. 프로그램 사용 중 불편한 점을 개선해 보고 싶은 분, 실패를 두려워하지 않는 분에게 추천. 코딩하면 쉽게 돈 벌 수 있다는 환상으로 시작하면 안 됨"

### 인터뷰 4: 1년 만에 100배 수익만든 50대 비전공자 (01:29:48~02:11:24)
- **배경**: 50대. 경영학 박사. 에스토니아 전자시민권으로 법인 보유 (2018년 설립)
- **프로젝트**: 통계 교육 플랫폼 구축
  - 100개 섹션, 9,000개 무료 통계 문제
  - Google Gemini 2.0 Flash로 통계 결과 AI 해석
  - Stripe 결제 연동 (기부 모델)
  - 100개 언어 서비스 (Google Translate)
  - 소멸 언어/방언/고어 복원 프로젝트 (UNESCO 소멸 언어 포함)
  - 에스토니아 법인 기반 논문 작성 지원
- **코딩 접근**: "HTML, CSS, JavaScript라는 단어도 아직 어색하지만 다 만들 수 있다. AI가 도와주니까 가능. 적어도 위험 요소나 사업을 하는 입장에서 아이디어가 있으면 실행해야 바뀐다"
- **인사이트**: "세상에 쉽게 버는 돈은 없다. 같은 강의를 들어도 받아들이는 케이스가 천차만별. 하지만 수준이 올라가면 코딩은 자동화가 있어서 집중적으로 하고 자기 시간을 확보할 수 있다는 장점이 매우 크다"

### 공통 교훈
1. **배경 무관**: 대리기사, 비전공자, 마케터, 50대 박사 — 누구나 가능
2. **커스터마이징 필수**: 같은 코드를 그대로 쓰면 차별화 없음 → 수익 없음
3. **비용 최소**: AI 구독료(월 20달러) 외에 사실상 비용 없음
4. **자동화 = 시간 확보**: 한 번 세팅하면 자동으로 수익 발생
5. **실패는 과정**: 해결 안 되면 드랍하고 다른 것 시도. 14시간 몰입하는 끈기

## 원문 인용
> "코딩은 일 지식 분야 중에서도 레벨이 가장 높은 학문이라고 생각해서 저는 절대 못 할 거라고 생각했고요" (대리기사 인터뷰)
> "투더제이 님은 코딩을 어렵게 알려주는 느낌이 아니라 코딩으로 센스 있게 돈을 많이 버는 능력 있는 형 같은 느낌" (대리기사 인터뷰)
> "무료 API를 프롬프트를 잘해서 사람이 쓴 것처럼 자동화를 돌리고, 여러 아이디를 돌리면서도 API 호출 비용이 나온 적이 한 번도 없고요" (비전공자 인터뷰)
> "커스터마이징 한 블로그는 하루 2,100명, 그대로 돌린 건 하루 50명" (비전공자 인터뷰)
> "14시간 동안 밥도 안 먹고 화장실도 안 가고 문제를 파고든 적도 있다" (마케터 인터뷰)
> "세상에 쉽게 버는 돈은 없다. 하지만 수준이 올라가면 코딩은 자동화가 있어서 자기 시간을 확보할 수 있다" (공통)
