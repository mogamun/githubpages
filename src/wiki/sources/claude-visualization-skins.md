# 클로드 100개 시각화 스킨 무료 배포

## 개요
AI Korea Community에서 배포한 Claude 시각화 프롬프트 템플릿. 글로벌 컨설팅 펌(Accenture, Deloitte, PwC, EY) 수준의 HTML 프레젠테이션 슬라이드를 생성하는 구조화된 프롬프트 + 100개 분류별 컬러 스킨.

## 핵심 내용
- **프롬프트 구조**: 디자인 사양(컬러 스킨, 레이아웃) + 콘텐츠 구성(요약, 데이터 시각화, 핵심 포인트) + 품질 체크리스트
- **100개 컬러 스킨** (10개 카테고리 × 10개):
  - 사업계획서(비즈니스, 미니멀, 테크, 서스테이너블, 럭셔리 등)
  - 명품 브랜드(버버리 체크, 샤넬 흑백, 구찌 빈티지 등)
  - 산업 분야(금융, 마케팅, 의료, IT, 교육 등)
  - 디자인 트렌드(사이키델릭, 글래스모피즘, 바이오필릭 등)
  - 국가별(북유럽, 미국, 모로코, 한국 한옥 등)
  - 럭셔리 자동차(벤츠, BMW, 페라리, 롤스로이스 등)
  - 아이스크림, 과일, 직업별, 영화/TV 시리즈
- **사용법**: 프롬프트 템플릿 복사 → 컬러 스킨 교체 → 기사 내용 입력 → Claude에 붙여넣기 → HTML 다운로드
