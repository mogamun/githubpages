# multiagent

## 개요
agentchattr(Python FastAPI)를 Hono + Vite + React + Tailwind 스택으로 컨버팅한 AI 에이전트 코디네이션 채팅 서버. tmux 기반 터미널 세션, 폴링 기반 채팅, 에이전트 등록/하트비트, Box 관리 시스템 포함.

## 기술 스택
- Hono (백엔드 프레임워크)
- Vite (빌드/개발 서버)
- React + Tailwind CSS (프론트엔드)
- TypeScript
- tmux (터미널 세션 관리)
- xterm.js (브라우저 터미널)
- Capacitor (Android/iOS 앱)
