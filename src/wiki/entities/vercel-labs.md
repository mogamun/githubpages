# Vercel Labs

## Identity
- **Type**: 기술 조직 (Vercel 실험실)
- **Parent**: Vercel (Next.js 개발사)
- **GitHub**: https://github.com/vercel-labs

## 주요 프로젝트
- **agent-browser**: AI 에이전트용 브라우저 자동화 CLI (30.8k stars, Rust 기반)
  - Vercel 생태계와 긴밀한 통합 (Vercel Sandbox에서 서버리스 실행 지원)

## Relevance
Vercel 생태계의 실험적 프로젝트를 통해 AI 에이전트 인프라를 확장. agent-browser는 MCP 없이 CLI 기반으로 브라우저 자동화를 수행하는 새로운 패러다임 제시.
