# Keygraph

## 정체
- **Type**: 보안 기술 기업
- **URL**: https://keygraph.io
- **제품**: Shannon (AI 침투테스트), Shannon Pro (통합 AppSec 플랫폼)

## 역할
AI 기반 애플리케이션 보안 테스트 도구 개발. Shannon은 화이트박스 AI 펜테스터, Shannon Pro는 SAST + SCA + secrets + 비즈니스 로직 테스트 + 자율 펜테스팅 통합 플랫폼. Pro는 CPG(Code Property Graph) 기반 정적-동적 상관 분석 제공.
