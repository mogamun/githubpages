# 소스놀이터

## 정체
한국어 AI 도구 실전 시연 유튜브 채널. 새로운 AI 도구와 기능을 직접 사용해 보고 결과물을 공개하는 리뷰어.

## 역할
- AI 이미지 생성 도구 비교 분석 (GPT 이미지 2.0, 나노바나나 프로, Gemini)
- 코덱스(Codex) CLI 실전 활용법 시연
- 바이브 코딩 워크플로우 소개
- 소놀봇 카톡 오픈채팅 운영 (피드백, 잡담)
- AI/바이브 코딩 오픈채팅 운영

## 특징
- GPT 이미지 2.0의 한국어 완벽 렌더링을 최초로 상세 시연
- 코덱스 CLI의 무제한 이미지 생성 설정법 공개
- PPT 스타일 대신 이미지 직접 생성 방식 선호
- 서포터즈: 오토 블로그 프로그램, 이미지 생성 솔루션(MCP 지원), 원격 지원

## 관련 프로젝트/맥락
- 코덱스 CLI를 활용한 자율 에이전트 시연
- GPT 5.4 Fast 모델 실전 사용
- 멀티모달 AI 분석 (메인보드, 3D 설계도, AI 변천사)
