# stashYoutube

## 개요
네트워크 폴더 기반 비디오 스트리밍 웹 애플리케이션. 로컬/네트워크 저장소의 영상을 브라우징하고 스트리밍 재생하며 썸네일 캡처, 파일 이동 기능 제공.

## 기술 스택
- Frontend: React 19, Vite 7, TypeScript 5.9, styled-components, react-router-dom
- Backend: Node.js, Express 4, fluent-ffmpeg
- 기타: HLS/MP4 스트리밍, 로그인 인증 (login.json)
