# 투더제이 (TTJ)

## 정체
- **Type**: 유튜브 채널 (코딩 수익화 : 투더제이 TTJ)
- **내용**: 바이브 코딩 입문 시리즈(Vol.1-3), 코딩 수익화 강의, UI/UX 쇼츠 팁
- **저서**: 『모두를 위한 진짜 쎈 바이브코딩 with 인스타그램 클론 코딩』
- **가이드북**: 바이브코딩 치트키 15 (ttj.kr/free 무료 배포)
- **부트캠프**: 17개 수익 파이프라인, 2년 수강기간 코딩 수익화 전문 부트캠프

## 역할
바이브 코딩 입문자를 위한 실전 가이드 제공. 비전공자/대리기사/마케터/50대 등 다양한 배경의 수강생 수익화 사례 증명. "AI를 잘 쓰는 것이 아니라 AI를 잘 조련하는" 방법론 체계화.

## 핵심 방법론
- **치트키 15**: PRD → 페르소나 → 기술 스택 고정 → 컨텍스트 관리 → 프롬프트 체이닝
- **학습 철학**: "대충 하시는 게 가장 중요" — 기술 부채는 나중에 해결
- **추천 스택**: Next.js + Supabase + Tailwind CSS (2026년)
- **도구 흐름**: "커서 AI로 코딩 기본 배우고 클로드코드로 완성"
