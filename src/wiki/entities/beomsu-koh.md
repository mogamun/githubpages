# 고범수 (Beomsu Koh)

## 정체
한국어 AI 에이전트 기술 리뷰어. Hermes Agent 공식 독스와 GitHub를 깊게 분석하여 기술 아키텍처를 정리하는 콘텐츠 크리에이터.

## 역할
- Hermes Agent 공식 독스 기반 기술 심층 분석
- "LLM + Harness" 멘탈 모델 제시
- 독스 읽기 방법론: "프로그램이면 파일에 저장되어 있을 것 → 파일시스템과 커맨드를 읽어라"
- Deep Research 활용법: 모르는 도메인은 프롬프트 구조화보다 독스 차근차근 읽기가 더 효과적
